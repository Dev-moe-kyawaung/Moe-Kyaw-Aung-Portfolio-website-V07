# Moe-Kyaw-Aung-Portfolio V7

Premium glasspunk single-page portfolio for **Moe Kyaw Aung** — Senior Android Developer.

## Features
- React + Vite
- Glasspunk transparent design
- Animated particle hero
- Animated counters
- Dark / Light mode
- Smooth scroll
- Mobile menu
- Carousel
- Pricing
- FAQ
- Lightbox gallery
- Social icons
- Google Maps embed
- Sticky CTA
- Back to top
- Newsletter form
- Contact form with validation

## GitHub Pages Deployment
1. Replace `Dev-moe-kyawaung` and `Moe-Kyaw-Aung-Portfolio-website-V07` in `package.json` and `vite.config.js`.
2. Install dependencies:
   ```bash
   npm install
npm run dev      # Start development server
npm run build    # Build for production
npm run preview  # Preview production build locally
npm run deploy   # Deploy to GitHub Pages
